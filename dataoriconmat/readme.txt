concno folder contains the original data of control group which receive hm4di virus during training and receive CNO injection during test recall.
conpbs folder contains the original data of control group which receive hm4di virus during training and receive PBS injection during test recall.
the each file in folder represent the data of a sigle type of behavior, like hit, miss, correct rejection(CR) or false alarm(FA).

con_CNOandPBS folder contains the merged data according data in concno folder and conpbs folder. The merge rule is merging the same behavior in concno and conpbs.
